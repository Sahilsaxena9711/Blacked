# Blacked

Dark Mode for React Apps

## Installation

```
npm install blacked or yarn add blacked
```

## Usage

-Import Blacked
```
 import Blacked from 'blacked';
```

-Wrap your component
```
<Blacked>
    <ComponentName />
</Blacked>
```

## Props

```
position : "top-right" || "top-left" || "bottom-right" || "bottom-left" 
//default is "top-right"

Example :-

<Blacked position={"top-right"}>
    <ComponentName />
</Blacked>
```
